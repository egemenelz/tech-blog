## Usage

After opening project with VisualStudio Code

````
npm i - will install all the needed dependecies
nodemon - will start the server
touch .env
    
    DB_NAME = 'database name here'
    DB_USER = 'root'
    DB_PW = 'password'
    SECRET= 'secret here'
    
mysql -u root -p
Enter your password:
    source db/schema.sql

````
